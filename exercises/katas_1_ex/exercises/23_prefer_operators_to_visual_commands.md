### Prefer operators to visual commands

Change the text in the tags to be upper-case.

```html
<a href="#">one</a>
<a href="#">two</a>
<a href="#">three</a>
```

This is what the HTML snippet should look like:

```html
<a href="#">ONE</a>
<a href="#">TWO</a>
<a href="#">THREE</a>
```

`gU{motion}` - visual mode equivalent in normal mode
